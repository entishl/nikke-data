# Lucky

这是一个用于管理游戏角色数据的全栈应用。

## 如何运行

1.  **构建并运行 Docker 容器:**

    ```bash
    docker-compose up --build
    ```

2.  **访问应用:**

    *   前端: [http://localhost:8080](http://localhost:8080)
    *   后端 API: [http://localhost:8000/docs](http://localhost:8000/docs)
